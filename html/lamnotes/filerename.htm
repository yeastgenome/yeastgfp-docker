(Successful with Red Hat 7.3).




the following builds the jar and installs it:
cd ~/javaWork/util; make; cd ..; jar cf util.jar util/*.class; cd util; make clean; sudo mv ../util.jar /usr/java/j2re1.4.1/lib/ext/

next, do the following to rename :'s to --
find . -type f | java util.stringReplace "\\\\" "\\\\\\\\" | java util.stringReplace " " "\\\\ " | java util.stringReplace ":" "\\\\:" | java util.stringReplace "'" "\\\\'" | xargs -i java util.fileCopy {} ":" "--"

And this to remove single quotes:
find . -type f | java util.stringReplace "\\\\" "\\\\\\\\" | java util.stringReplace " " "\\\\ " | java util.stringReplace ":" "\\\\:" | java util.stringReplace "'" "\\\\'" | xargs -i java util.fileCopy {} "'" ""
