to install a java utility

L. Gerke - 1/2003

(Successful with Red Hat 7.3).



Imagine you want to have a utility called escape

it's in a package called util.  create the dir structure ./util/escape.java

the .java must start with: package util;

compile it, and from .: jar cf util.jar util/escape.class
Now put util.jar into the classpath:
either invoke by: java -classpath ./././util.jar util.escape
or: export CLASSPATH=$CLASSPATH:./././util.jar
and java util.jar
-Xrunhprof:cpu=samples,depth=20
