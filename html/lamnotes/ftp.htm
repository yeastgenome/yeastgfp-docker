Configuring wu-ftpd

(Successful on RedHat 7.1).
  1. As root, edit /etc/xinetd.d/wu-ftpd.
  2. Switch disable to no.
  3. Restart xinetd with /sbin/service xinetd restart.
(* Remember that ftp is unencrypted and completely unsecure. The better alternative is to run sshd which has sftp piggybacked onto it.) Notes: For anonymous FTP, install the anonftp RPM package.
