Installing ImageMagick on Linux

F. Lam - 6/5/2002

(Successful on Red Hat 7.3).

  1. Download the latest ImageMagick source (tar.gz).
  2. As root: 
    
tar -xzf [ImageMagick]
cd [ImageMagick]
./configure | less (examine output to make sure everything is ok)
make
make install
    The necessary JPEG and TIFF libraries should already be installed by Red Hat.

Using ImageMagick

Check out the Mac OS X instructions for ImageMagick commands.
