(Successful with Red Hat 7.3). 


install the rpms for sdk and jre
cp ****/mysql-connector-java-2.0.14-bin.jar /usr/java/j2re1.4.1_01/lib/ext/

export PATH=$PATH:/usr/java/j2re1.4.1/bin:/usr/java/j2sdk1.4.1/bin
export JAVA_HOME=/usr/java
export CLASSPATH=.:/usr/java/j2re1.4.1/lib/ext/mysql-connector-java-2.0.14-bin.jar

in mysql:
GRANT ALL PRIVILEGES ON gfp.* to 'monty'@'localhost.localdomain' identified by 'beerkeg'

Now access the db like:

Class.forName("com.mysql.jdbc.Driver").newInstance();
	    java.sql.Connection conn;
	    conn = DriverManager.getConnection("jdbc:mysql://localhost/gfp", "monty", "beerkeg");
	    java.sql.Statement stmt = conn.createStatement();
	    java.sql.ResultSet result = stmt.executeQuery("SELECT * FROM orfs ORDER BY orfid");
